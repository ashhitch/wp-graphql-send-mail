This plugin enables to send email via WPGraphQL.

== Installation ==

1. Install & activate [WPGraphQL](https://www.wpgraphql.com/)
2. Install & activate this plugin to the `/wp-content/plugins/` directory


== Usage ==

```
mutation SEND_EMAIL {
  sendEmail(
    input: {
      to: "test@test.com"
      from: "test@test.com"
      subject: "test email"
      body: "test email"
      clientMutationId: "test"
    }
  ) {
    origin
    sent
    message
  }
}

```
